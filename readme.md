# Where Is My Money - Website Plan

## Project Direction (based on your research)
This website presents **Where Is My Money** as a financial transparency tool for independent artists and labels.

Core problem from your research:
- Artists get paid from many places (streaming, royalties, live, collaborations, fan support).
- Payment data is fragmented across platforms.
- Artists struggle to answer basic questions: what was paid, when, and from where.

Core website goal:
- Explain the problem clearly.
- Show the product concept simply.
- Build trust and collect early interest.

## Audience
- Independent singers and artists
- Small independent labels
- Artist managers and creative teams

## Website Scope (for now)
In scope:
- Marketing website (not full software product)
- Product story, benefits, and feature concept
- Waitlist/contact capture

Out of scope:
- Real payment integrations
- Live dashboards from real data sources
- Full web app/backend

## Delivery Method
Work in small stages so you can manage and approve each step.
Each stage is intentionally limited and should be reviewed before starting the next one.
